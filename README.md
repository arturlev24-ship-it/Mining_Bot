# 🌟 Telegram Mining Bot

Телеграм бот для майнинга звезд с реферальной системой и мультиязычной поддержкой.

## ✨ Возможности

- ⛏ **Майнинг звезд** - зарабатывайте звезды Telegram
- 👥 **Реферальная система** - получайте бонусы за друзей
- 🌍 **Мультиязычность** - русский и английский языки
- 📤 **Вывод средств** - от 100 звезд
- 🎁 **Рекламная система** - показ рекламы каждые 8 часов
- ⏰ **Автоотключение** - майнинг отключается каждые 3 часа
- 💾 **Автосохранение** - данные сохраняются каждые 5 минут
